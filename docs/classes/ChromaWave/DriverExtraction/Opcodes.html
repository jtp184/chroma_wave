module ChromaWave::DriverExtraction::Opcodes

  1. lib/chroma_wave/driver_extraction/opcodes.rb

Sentinel opcodes and lookup tables for driver config extraction.

Sentinel opcodes match the constants defined in chroma_wave.h. Lookup tables map Ruby symbols to their C enum/flag names.
