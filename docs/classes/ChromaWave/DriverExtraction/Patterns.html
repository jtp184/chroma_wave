module ChromaWave::DriverExtraction::Patterns

  1. lib/chroma_wave/driver_extraction/patterns.rb

Regex patterns for parsing Waveshare vendor C source files.

All patterns are constants to avoid repeated compilation. Complex patterns use /x mode for readability.
